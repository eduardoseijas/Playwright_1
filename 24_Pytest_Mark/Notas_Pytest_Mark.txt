el comando para lanzarlo, le agregas un -s -v por ejemplo 

python -m pytest .\24_Pytest_Mark\Test_Pytest_Mark_1.py -s -v

Este es para que asi la prueba tenga un error pueda correr, y colocas esto
encima de la definicion del test
